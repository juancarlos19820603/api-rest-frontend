# API Frontend Client

Frontend desarrollado en React que consume una API REST construida con Node.js, Express y MongoDB.

## Descripción

Este proyecto implementa la interfaz de usuario para autenticación y gestión de usuarios.  
Se conecta a un backend que maneja JWT, roles y validaciones seguras.

El objetivo del proyecto es demostrar integración Full Stack y buenas prácticas de arquitectura.

---

## Tecnologías utilizadas

- React
- JavaScript (ES6+)
- Axios
- React Router DOM
- CSS
- JWT (JSON Web Token)
- Context API (si lo usas)
